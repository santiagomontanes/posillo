# Casos de uso
1. Login por email/password con roles ADMIN/SELLER.
2. Vender producto, descontar stock y generar factura.
3. CRUD inventario (eliminación solo ADMIN).
4. Registrar gastos y consultarlos por fechas.
5. Abrir/cerrar caja con cálculo esperado.
6. Visualizar reportes y top productos.
7. Exportar/restaurar backups (restore solo ADMIN).
