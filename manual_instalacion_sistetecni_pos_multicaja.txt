🖥 MANUAL DE INSTALACIÓN

SISTETECNI POS – MODO MULTICAJA (MYSQL)

⸻

📌 ESCENARIO
	•	Caja 1 = Servidor (tiene MySQL instalado)
	•	Caja 2 = Cliente (NO necesita MySQL)
	•	Ambas deben estar en la misma red
	•	Base de datos se crea automáticamente con script SQL
	•	Conexión se configura con window.api.mysql.setConfig

⸻

🔹 PARTE 1 – CONFIGURAR EL SERVIDOR (CAJA 1)

⸻

1️⃣ Instalar MySQL Server

Instalar:

MySQL Server 8.0

Durante instalación:
	•	Guardar contraseña de root
	•	Dejar puerto 3306
	•	Modo Standalone

Verificar que el servicio esté activo:

Win + R →

services.msc
MySQL80 DEBE DECIR RUNNINGG

CAPIAR ARCHIVO SETUP_SISTETECNI_POS.SQL
EJECUTAR DESDE CMD :
mysql -u root -p < C:\RUTA\setup_sistetecni_pos.sql
CONTRASEÑA DEL ROOT
ABRIR :C:\ProgramData\MySQL\MySQL Server 8.0\my.ini
VERIFICAR
[mysqld]
bind-address=0.0.0.0
port=3306
GUARDAR Y REINCIAE ELS ERVICIO CON :
net stop MySQL80
net start MySQL80
EJECUTAR COMO ADMINISTRADO
netsh advfirewall firewall add rule name="MySQL 3306" dir=in action=allow protocol=TCP localport=3306
Y ANOTAR LA IP DE IPCONFIG

Y AHORA ABRIR DEVTOOLS:await window.api.mysql.setConfig({
  host: "192.168.101.83",
  port: 3306,
  user: "pos_user",
  password: "admin",
  database: "sistetecni_pos"
});

await window.api.mysql.test();
